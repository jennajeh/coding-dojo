#### 1. 이해

#### 2. 계획

- 맞춘 번호 최대값 = 맞춘 번호 + 0의 개수 (0이 모두 맞았다고 가정)
- 맞춘 번호 최소값 = 맞춘 번호 (0이 있을 때 모두 틀렸다고 가정)
- 맞춘 개수에 따라 최고 순위와 최저 순위 계산
